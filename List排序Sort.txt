  1:换 -1:不换 0:相等,不换

	第一种写法:
        sortList.Sort(
            (a,b) => 
            {
                if (a.Value > b.Value)
                    return 1;
                else if (a.Value < b.Value)
                    return -1;
                else
                    return 0;
            }
            );
	
	第二种写法: 推荐使用中CompareTo比较
	sortList.Sort((a,b)=>{return a.CompareTo(b)})
